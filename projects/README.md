## 🌟 What is OSMEA Projects?

**OSMEA Projects** is a curated collection of production-ready Flutter applications and development tools that demonstrate the power and versatility of the OSMEA ecosystem. Each project serves a specific purpose and showcases different aspects of modern Flutter development, from UI component libraries to complete e-commerce solutions.

### ✨ **Key Features**

- **🎨 Complete Applications** - Full-featured Flutter apps ready for production
- **🛠️ Development Tools** - Interactive playgrounds and testing utilities
- **📱 Cross-Platform** - iOS, Android, and Web support across all projects
- **🏗️ Modular Architecture** - Clean, scalable, and maintainable code structure
- **📚 Learning Resources** - Comprehensive examples and documentation

## 🚀 Quick Start

### **📋 Prerequisites**

- **Flutter SDK** (3.7.2 or higher)
- **Dart SDK** (3.6.1 or higher)
- **Android Studio** or **VS Code** with Flutter extensions
- **Git** for version control

### **📦 Installation**

1. **Clone the repository:**
   ```bash
   git clone https://github.com/masterfabric-mobile/osmea.git
   cd osmea
   ```

2. **Navigate to a project:**
   ```bash
   cd projects/[project_name]
   ```

3. **Install dependencies:**
   ```bash
   flutter pub get
   ```

4. **Generate code (if needed):**
   ```bash
   flutter packages pub run build_runner build
   ```

5. **Run the project:**
   ```bash
   # Run on web (recommended for demos)
   flutter run -d chrome
   
   # Run on mobile
   flutter run
   
   # Run on specific device
   flutter run -d <device-id>
   ```

### **🌐 Web Deployment**

For production deployment:

```bash
# Build for web
flutter build web --release

# Deploy to your hosting service
# The build/web directory contains all static files
```
